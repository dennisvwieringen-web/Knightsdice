# Knight's Dice

Knight's Dice is een middeleeuws dobbelspel met quests, skins, medailles, skinpakjes en eindbazen.

## Spelen

Open `index.html` lokaal, of publiceer deze repository met GitHub Pages.

Op Android:

1. Open de GitHub Pages-link in Chrome.
2. Tik op het menu.
3. Kies `Toevoegen aan startscherm`.

Daarna werkt Knight's Dice als een app op je telefoon.

## Bestanden

- `index.html` - de pagina
- `styles.css` - de vormgeving
- `game.js` - het spel
- `manifest.webmanifest` - Android/app-instellingen
- `service-worker.js` - offline/app-cache
